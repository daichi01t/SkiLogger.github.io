SkiLogger Privacy Policy
プライバシーポリシー

Effective date / 施行日: 2026-04-29

1. Overview / 概要

日本語

このプライバシーポリシーは、Daichi Takikawa（以下「サービス提供者」）が提供するモバイルアプリ「SkiLogger」（以下「本アプリ」）に適用されます。本アプリは、スキーの記録、写真の保存、共有用画像の作成を目的として提供されます。

本アプリは、ユーザーの記録を主に端末内に保存します。サービス提供者は、本アプリを通じてユーザーの個人情報を外部サーバーに収集・送信することを目的としていません。

English

This Privacy Policy applies to the SkiLogger mobile application (the “Application”) provided by Daichi Takikawa (the “Service Provider”). The Application is intended to help users record skiing activities, save photos, and create images for sharing.

The Application primarily stores user records on the user’s device. The Service Provider does not intend to collect or transmit users’ personal information to an external server through the Application.

2. Information Stored or Used / 保存または利用される情報

日本語

本アプリでは、ユーザーが入力または選択した以下の情報を保存する場合があります。

  • スキー場、日付、雪質、天気、混雑状況などのログ情報
  • メモなど、ユーザーが任意で入力した内容
  • ユーザーが選択または撮影した写真
  • プロフィール名、イニシャル、アバター設定
  • バックアップやインポートのためにユーザーが選択したデータ

これらの情報は、本アプリの機能を提供するために利用されます。

English

The Application may store the following information entered or selected by the user:

  • Log information such as ski resort, date, snow condition, weather, and crowd level
  • Optional user-entered content such as notes
  • Photos selected or captured by the user
  • Profile name, initials, and avatar settings
  • Data selected by the user for backup or import

This information is used to provide the features of the Application.

3. Photos / 写真

日本語

本アプリは、ユーザーが選択または撮影した写真をスキー記録に添付するために利用します。写真は、ユーザーが追加した場合にのみ本アプリ内に保存されます。

本アプリは、ユーザーの写真ライブラリ全体を収集したり、サービス提供者の外部サーバーへ写真を送信したりしません。

English

The Application uses photos selected or captured by the user to attach them to skiing records. Photos are stored in the Application only when the user adds them.

The Application does not collect the user’s entire photo library or transmit photos to an external server operated by the Service Provider.

4. Location / 位置情報

日本語

本アプリは、チェックイン時に現在地に近いスキー場を提案するため、ユーザーの許可を得たうえで位置情報を利用する場合があります。

位置情報は端末上で近隣のスキー場を判定する目的で使用されます。サービス提供者は、本アプリを通じて正確な位置情報を外部サーバーに収集・送信しません。

English

The Application may use location information, with the user’s permission, to suggest ski resorts near the user during check-in.

Location information is used on the device to determine nearby ski resorts. The Service Provider does not collect or transmit precise location information to an external server through the Application.

5. iCloud Backup and Import / iCloudバックアップとインポート

日本語

本アプリは、ユーザーの操作によりバックアップファイルの作成やインポートを行う機能を提供する場合があります。ユーザーがiCloud Driveなどを保存先として選択した場合、そのデータはAppleが提供するサービス上に保存されることがあります。

iCloudに保存されたデータの取り扱いは、Appleの規約およびプライバシーポリシーに従います。

English

The Application may provide features that allow users to create and import backup files. If the user chooses iCloud Drive or a similar location as the storage destination, the data may be stored using services provided by Apple.

Data stored in iCloud is handled according to Apple’s terms and privacy policy.

6. Third Parties and AI / 第三者サービスおよびAI

日本語

本アプリは、サービス提供者が運営する外部分析サービスにユーザーデータを送信することを目的としていません。また、本アプリはユーザーデータを処理するために人工知能（AI）技術を使用しません。

ただし、App Store、iCloud、iOSなど、Appleが提供するプラットフォーム機能の利用に伴うデータ処理については、Appleのポリシーが適用される場合があります。

English

The Application is not intended to transmit user data to external analytics services operated by the Service Provider. The Application does not use Artificial Intelligence (AI) technologies to process user data.

However, data processing related to Apple platform features such as the App Store, iCloud, and iOS may be subject to Apple’s policies.

7. Data Retention and Deletion / データの保持と削除

日本語

本アプリ内のログ、写真、プロフィール情報は、ユーザーが本アプリを利用している間、端末内に保存されます。ユーザーは、アプリ内の削除機能または端末のアンインストール機能により、保存されたデータを削除できます。

バックアップファイルを作成した場合、そのファイルの管理と削除はユーザー自身で行う必要があります。

English

Logs, photos, and profile information in the Application are stored on the user’s device while the user uses the Application. Users can delete stored data using the deletion features in the Application or by uninstalling the Application from their device.

If the user creates backup files, the user is responsible for managing and deleting those files.

8. Security / セキュリティ

日本語

サービス提供者は、ユーザーの情報を適切に取り扱うよう努めます。ただし、インターネット上の通信や電子的な保存方法について、完全な安全性を保証することはできません。

English

The Service Provider strives to handle user information appropriately. However, no method of transmission over the Internet or method of electronic storage can be guaranteed to be completely secure.

9. Children / 子どものプライバシー

日本語

本アプリは、13歳未満の子どもから故意に個人情報を収集することを目的としていません。13歳未満の子どもが個人情報を提供した可能性があると考えられる場合は、下記連絡先までご連絡ください。

English

The Application is not intended to knowingly collect personal information from children under the age of 13. If you believe that a child under 13 may have provided personal information, please contact the Service Provider using the contact information below.
